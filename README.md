# 🚖 Uber Power BI Dashboard

## 📌 Project Overview

This project is an **Uber Data Analysis Dashboard** built using **Power BI** to analyze ride performance, booking trends, trip insights, ratings, and business KPIs. The dashboard helps visualize Uber ride data through interactive reports and meaningful insights.

## 📊 Dashboard Features

* **Trip Analysis** – Analyze ride trends and trip performance.
* **Booking Insights** – Monitor ride booking patterns.
* **Performance KPIs** – View important metrics in cards and visuals.
* **Ratings Analysis** – Track customer ride ratings.
* **Distance & Travel Metrics** – Understand travel behavior and distance covered.
* **Interactive Filters (Slicers)** – Dynamic filtering for better analysis.

## 🛠️ Tools & Technologies Used

* **Power BI**
* **Data Visualization**
* **DAX (Data Analysis Expressions)**
* **Data Cleaning & Transformation**
* **Business Intelligence Reporting**

## 📈 Key KPIs Used
